# 小白写的第一个正经的程序
## 写这个简单程序的原因
 - 我要看**色色**
 - 好的**色色**不是很好找
 - TG上有**色色！！**
 - 得搞一个批量下载**色色**的东西
 - 哦，原来自带啊
 - 啊，这导出的东西有点多啊，好多没用的

#### 于是我写了这个没甚么大用的东西

程序会自动删除在给定目录下的，符合给定文件名后几位字符串的文件
**也就是说这个程序可以进行某文件夹下所有同后缀的文件，不止可以删TG里下的图片缩略图，例如还可以删百度网盘下载时生成的缓存文件**
